The Vertebral Heart Size (VHS) metric is a com-monly used technique to evaluate cardiac enlargement in dogsusing X-ray images. However, conventional VHS estimation needsmanual annotation of anatomical landmarks, making the processtime-intensive and leading to human error. In this project, weintroduced an automated solution for VHS computation usinga compact convolutional neural network (CNN) that detectssix anatomical keypoints from X-ray images. The predictedpoints are then used to calculate VHS based on the established6(AB + CD)/EF formula. Our custom architecture, TinyVHSKey-pointModel, is built for fast execution while maintaining betteraccuracy, making it best for veterinary use. Training and testingwere performed on the Dog Heart VHS data set, and themodel output (a CSV ﬁle) was validated using external software,achieving a test set accuracy of 76 percent. The results areformatted to merge with clinical tools, offering a reliable andeffective alternative to manual evaluation
